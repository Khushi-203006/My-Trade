# My Trade

Stock Trading Strategy Project

## Objective

Build a data-driven stock trading system that:

- Downloads NSE stock data
- Processes and cleans data
- Creates trading features
- Uses machine learning for prediction
- Performs backtesting
- Automates daily execution
- Provides dashboards and reports

## Project Structure

data/          - Raw and processed data
src/           - Source code
reports/       - Generated reports
docs/          - Documentation

## Technology Stack

- Python 3.11
- Pandas
- NumPy
- Scikit-Learn
- Git
- GitHub
- Power BI
